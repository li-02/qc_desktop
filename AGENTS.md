# Repository Instructions

This repository uses `.github/instructions/eco-ui-style.instructions.md` as the single source of truth for frontend UI design rules.

When changing any file under `frontend/**`, read and follow `.github/instructions/eco-ui-style.instructions.md` before editing. Do not copy those rules into this file; keep this file as a thin pointer so Codex, Claude Code, GitHub Copilot, Cursor, and other agents all converge on the same UI standard.
