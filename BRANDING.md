# QC Studio Branding

## Product Name

- English name: QC Studio

## Icon

The app icon is stored at `public/icons/app-icon.svg`, with generated runtime assets in the same folder.

The icon is a red seed growing into two green leaves. It keeps the friendly startup-icon direction while redrawing the artwork as a project-owned, crisp app icon suitable for installers, taskbars, and small favicon sizes.
