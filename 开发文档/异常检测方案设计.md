# 异常检测方案设计文档

## 1. 概述

本文档定义了 QC Desktop 应用的异常检测模块设计方案，涵盖阈值过滤、传统统计方法、以及未来 AI 模型的扩展能力。

---

## 2. 你的方案评估

### 2.1 优点

| 设计点                  | 评价                     |
| -------------------- | ---------------------- |
| **阈值上下限过滤**          | ✅ 基础且必要，符合通量数据 QC 标准流程 |
| **三级作用域（数据集/项目/应用）** | ✅ 灵活，支持继承与覆盖，减少重复配置    |
| **Despiking 等传统方法**  | ✅ 通量数据常用，如 MAD、滑动窗口等   |
| **模型兼容性预留**          | ✅ 前瞻性设计，便于后期扩展         |

### 2.2 需要补充的考虑

| 补充点                   | 说明                                   |
| --------------------- | ------------------------------------ |
| **多方法组合与执行顺序**        | 用户可能需要先 despiking 再阈值过滤，需支持 Pipeline |
| **异常值标记 vs 删除 vs 替换** | 不同场景需求不同，应支持多种处理方式                   |
| **时间窗口上下文**           | 通量数据的异常往往与时间相关（如夜间 CO2 通量）           |
| **多变量联合检测**           | 某些异常需要结合多个指标判断（如 u* 过滤）              |
| **可视化与人工复核**          | 异常检测结果需要可视化确认，支持人工修正                 |
| **检测结果可追溯**           | 记录每个异常值的检测方法、时间、操作人                  |
| **批量操作与撤销**           | 支持批量应用/撤销异常检测规则                      |

```python
from abc import ABC, abstractmethod
from typing import List, Dict, Any
import pandas as pd
import numpy as np

class OutlierDetector(ABC):
    """异常检测器基类"""

    method_id: str
    method_name: str
    requires_training: bool = False

    @abstractmethod
    def detect(self, series: pd.Series, **params) -> np.ndarray:
        """
        检测异常值

        Args:
            series: 输入数据序列
            **params: 方法参数

        Returns:
            布尔数组，True 表示异常
        """
        pass

    def get_default_params(self) -> Dict[str, Any]:
        """返回默认参数"""
        return {}

    def validate_params(self, params: Dict[str, Any]) -> bool:
        """验证参数有效性"""
        return True
```

### 10.3 具体实现示例

```python
# python/outlier_detection/statistical.py

class MADDespiking(OutlierDetector):
    """MAD Despiking 方法"""

    method_id = 'DESPIKING_MAD'
    method_name = 'MAD Despiking'

    def detect(self, series: pd.Series, 
               window_size: int = 13,
               threshold: float = 5.2,
               max_iterations: int = 10) -> np.ndarray:

        outliers = np.zeros(len(series), dtype=bool)
        data = series.copy()

        for iteration in range(max_iterations):
            # 计算滑动窗口中位数
            rolling_median = data.rolling(
                window=window_size, center=True, min_periods=1
            ).median()

            # 计算 MAD
            deviation = np.abs(data - rolling_median)
            mad = deviation.rolling(
                window=window_size, center=True, min_periods=1
            ).median()

            # 标记异常
            mad_score = deviation / (mad * 1.4826 + 1e-10)
            new_outliers = mad_score > threshold

            if not new_outliers.any():
                break

            outliers |= new_outliers
            data = data.mask(new_outliers)

        return outliers
```
