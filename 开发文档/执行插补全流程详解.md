## 1. 流程概览

执行插补是一个**多层异步操作**，涉及前端 Vue 组件、Electron IPC 通信、后端控制器、服务层和数据库操作。

**整体流程图**：


## 2. 前置条件

### 2.1 必需数据

在执行插补前，系统需要具备以下数据：

| 数据项 | 来源 | 说明 |
|--------|------|------|
| `datasetId` | `props.datasetInfo.id` | 当前数据集 ID |
| `versionId` | `currentVersion.value.id` | 当前选中的数据版本 ID |
| `methodId` | `selectedMethodId.value` | 用户选择的插补方法 ID |
| `targetColumns` | 用户选择或自动获取 | 需要插补的列名数组 |
| `params` | `paramValues.value` | 方法参数配置 |

### 2.2 数据库中必须存在的记录

```sql
-- 1. 数据集记录
SELECT * FROM biz_dataset WHERE id = :datasetId AND is_del = 0;

-- 2. 版本记录（包含数据文件路径）
SELECT * FROM biz_dataset_version WHERE id = :versionId AND is_del = 0;

-- 3. 插补方法配置
SELECT * FROM conf_imputation_method WHERE method_id = :methodId AND is_available = 1;
```

### 2.3 前端状态检查

```typescript
// GapFillingPanel.vue 中的 canExecute 计算属性
const canExecute = computed(() => {
```

```typescript
// 第 509-585 行
const executeImputation = async () => {
  // ========== 步骤 1: 校验前置条件 ==========
  if (!canExecute.value || !selectedMethodId.value || !currentVersion.value) return;

  try {
    // ========== 步骤 2: 初始化执行状态 ==========
    isExecuting.value = true;
    progressInfo.value = {
      resultId: 0,
      progress: 0,
      stage: 'preparing',
      message: '准备数据...',
    };
    executionLogs.value = [];

    // ========== 步骤 3: 构建目标列数组 ==========
    const targetCols = columnSelectionMode.value === 'all' 
      ? availableColumns.value.map(c => c.name)  // 全部列模式：获取所有列名
      : selectedColumns.value;                    // 手动模式：使用用户选择的列

    // ========== 步骤 4: 显示开始通知 ==========
    ElNotification({
      title: "开始插补",
      message: `正在使用 ${getMethodName(selectedMethodId.value)} 方法进行插补...`,
      type: "info",
      duration: 3000,
    });

    // ========== 步骤 5: 二次校验版本 ID ==========
    if (!currentVersion.value?.id) {
      ElMessage.error('无法获取当前数据版本ID，请尝试重新选择版本或刷新页面');
      isExecuting.value = false;
      return;
    }

    // ========== 步骤 6: 构建请求体 ==========
    const payload = {
      datasetId: parseInt(String(props.datasetInfo?.id || '0')),
      versionId: currentVersion.value.id,
      methodId: selectedMethodId.value,
      targetColumns: toRaw(targetCols),   // 使用 toRaw 去除 Vue 响应式包装
      params: toRaw(paramValues.value)    // 使用 toRaw 去除 Vue 响应式包装
    };
    
    console.log('[GapFillingPanel] Debug Execution Payload:', payload);

    // ========== 步骤 7: 调用后端 API ==========
    const result = await window.electronAPI.invoke(
      API_ROUTES.IMPUTATION.EXECUTE,  // 'imputation:execute'
      payload
    );

    // ========== 步骤 8: 处理执行结果 ==========
    if (result.success) {
      ElNotification({
        title: "插补计算完成",
        message: "请预览结果并选择保存方式",
        type: "success",
        duration: 5000,
      });
      
      // 进入预览模式
      currentResultId.value = result.resultId;
      isPreviewing.value = true;
      isExecuting.value = false;
      
      // 切换到结果视图
      viewResult({ 
        id: result.resultId, 
        methodId: selectedMethodId.value,
        status: 'COMPLETED' 
      } as any);

      // 刷新历史记录
      await loadHistory();
      emit("refresh");
    } else {
      throw new Error(result.error || '执行失败');
    }

  } catch (error: any) {
    console.error("插补处理失败:", error);
    ElMessage.error(error.message || "插补处理失败，请重试");
    addLog('error', error.message || "插补处理失败");
    isExecuting.value = false;
  }
};
```

```typescript
// 第 143-164 行
private async executeImputation(
  request: ExecuteImputationRequest,
  event: Electron.IpcMainInvokeEvent
): Promise<ExecuteImputationResponse> {
  try {
    // ========== 步骤 1: 创建进度回调函数 ==========
    // 这个回调函数会在服务层执行过程中被多次调用
    // 每次调用都会通过 IPC 将进度事件发送到渲染进程（前端）
    const progressCallback = (progressEvent: ImputationProgressEvent) => {
      try {
        // 使用 JSON 序列化确保对象可被克隆（IPC 要求）
        // 这会移除 undefined 值，避免序列化错误
        const serializableEvent = JSON.parse(JSON.stringify(progressEvent));
        
        // 通过 IPC 发送进度事件到前端
        // 前端通过 window.electronAPI.on('imputation:progress', ...) 监听
        event.sender.send('imputation:progress', serializableEvent);
      } catch (err) {
        console.error('发送进度事件失败:', err);
      }
    };

    // ========== 步骤 2: 调用服务层执行插补 ==========
    const result = await this.service.executeImputation(request, progressCallback);
    
    // ========== 步骤 3: 返回执行结果 ==========
    return result;
  } catch (error: any) {
    return { success: false, error: error.message };
  }
}
```

```typescript
// 第 87-218 行
async executeImputation(
  request: ExecuteImputationRequest,
  progressCallback?: ProgressCallback
): Promise<ExecuteImputationResponse> {
  const startTime = Date.now();

  // ==================== 阶段 1: 验证版本 ====================
  const version = this.db
    .prepare('SELECT * FROM biz_dataset_version WHERE id = ?')
    .get(request.versionId) as DatasetVersion | undefined;
  
  if (!version) {
    throw new Error(`版本不存在: ${request.versionId}`);
  }

  // ==================== 阶段 2: 读取数据文件 ====================
  // 从版本记录中获取文件路径，读取 CSV 数据
  const data = await this.readDataFile(version.file_path);
  if (!data || data.length === 0) {
    throw new Error('数据为空');
  }

  // ==================== 阶段 3: 处理目标列 ====================
  // 获取数据中所有可用的列名
  const availableColumns = Object.keys(data[0]);
  // 如果没有指定目标列，则使用所有列
  const targetColumns = request.targetColumns || availableColumns;

  // ==================== 阶段 4: 创建结果记录 ====================
  // 在数据库中创建一条状态为 PENDING 的记录
  const resultId = this.repository.createResult({
    datasetId: request.datasetId,
    versionId: request.versionId,
    methodId: request.methodId,
    targetColumns: targetColumns,
    methodParams: request.params,
  });

  // 注册进度回调
  if (progressCallback) {
    this.progressCallbacks.set(resultId, progressCallback);
  }

  try {
    // ==================== 阶段 5: 更新状态为运行中 ====================
    this.repository.updateResultStatus(resultId, 'RUNNING');
    this.emitProgress(resultId, {
      resultId,
      stage: 'preparing',
      progress: 0,
      message: '正在准备数据...',
    });

    // ==================== 阶段 6: 验证目标列有效性 ====================
    const invalidColumns = targetColumns.filter(col => !availableColumns.includes(col));
    if (invalidColumns.length > 0) {
      throw new Error(`无效的列: ${invalidColumns.join(', ')}`);
    }

    // ==================== 阶段 7: 执行插补（核心） ====================
    this.emitProgress(resultId, {
      resultId,
      stage: 'imputing',
      progress: 20,
      message: '正在执行插补...',
    });

    // 调用 performImputation 执行实际的插补算法
    const { imputedData, details, columnStats } = await this.performImputation(
      resultId,
      data,
      targetColumns,
      request.methodId,
      request.params || {},
      (progress, message, currentColumn) => {
        // 进度回调：20% - 80% 区间用于插补过程
        this.emitProgress(resultId, {
          resultId,
          stage: 'imputing',
          progress: 20 + progress * 0.6,
          message,
          currentColumn,
        });
      }
    );

    // ==================== 阶段 8: 保存详情和统计 ====================
    this.emitProgress(resultId, {
      resultId,
      stage: 'saving',
      progress: 80,
      message: '正在保存插补结果...',
    });

    // 批量插入插补详情
    if (details.length > 0) {
      this.repository.insertDetails(details);
    }

    // 插入列统计并累计总数
    let totalMissing = 0;
    let totalImputed = 0;
    for (const stat of columnStats) {
      this.repository.insertColumnStat(stat);
      totalMissing += stat.missingCount;
      totalImputed += stat.imputedCount;
    }

    // ==================== 阶段 9: 更新结果统计 ====================
    const executionTimeMs = Date.now() - startTime;
    this.repository.updateResultStats(resultId, {
      totalMissing,
      imputedCount: totalImputed,
      imputationRate: totalMissing > 0 ? totalImputed / totalMissing : 0,
      executionTimeMs,
    });

    // ==================== 阶段 10: 更新状态为完成 ====================
    this.repository.updateResultStatus(resultId, 'COMPLETED');
    this.emitProgress(resultId, {
      resultId,
      stage: 'saving',
      progress: 100,
      message: '插补完成',
    });

    // ==================== 返回成功响应 ====================
    return {
      success: true,
      resultId,
      message: `成功插补 ${totalImputed} 个缺失值`,
    };
  } catch (error: any) {
    // 出错时更新状态为失败
    this.repository.updateResultStatus(resultId, 'FAILED', error.message);
    return {
      success: false,
      resultId,
      error: error.message,
    };
  } finally {
    // 清理进度回调
    this.progressCallbacks.delete(resultId);
  }
}
```

#### 3.3.1 核心插补函数 `performImputation()`

```typescript
// 第 223-364 行
private async performImputation(
  resultId: number,
  data: Record<string, any>[],
  targetColumns: string[],
  methodId: ImputationMethodId,
  params: Record<string, any>,
  onProgress: (progress: number, message: string, currentColumn?: string) => void
): Promise<{
  imputedData: Record<string, any>[];
  details: Array<{...}>;
  columnStats: Array<{...}>;
}> {
  const details: Array<{...}> = [];
  const columnStats: Array<{...}> = [];
  const totalColumns = targetColumns.length;

  // ========== 遍历每一列进行插补 ==========
  for (let colIndex = 0; colIndex < targetColumns.length; colIndex++) {
    const columnName = targetColumns[colIndex];
    const progress = (colIndex / totalColumns) * 100;
    onProgress(progress, `正在处理列: ${columnName}`, columnName);

    // ---- 步骤 A: 提取列数据 ----
    const columnData = data.map(row => row[columnName]);
    
    // ---- 步骤 B: 转换为数值，处理空值 ----
    const numericData = columnData.map(v =>
      v === null || v === undefined || v === '' || Number.isNaN(Number(v)) 
        ? null 
        : Number(v)
    );

    // ---- 步骤 C: 找出缺失值索引和有效值 ----
    const missingIndices: number[] = [];
    const validValues: number[] = [];
    numericData.forEach((v, i) => {
      if (v === null) {
        missingIndices.push(i);
      } else {
        validValues.push(v);
      }
    });

    // ---- 步骤 D: 如果无缺失值，跳过 ----
    if (missingIndices.length === 0) {
      columnStats.push({
        resultId,
        columnName,
        missingCount: 0,
        imputedCount: 0,
        imputationRate: 0,
      });
      continue;
    }

    // ---- 步骤 E: 计算插补前统计 ----
    const meanBefore = this.calculateMean(validValues);
    const stdBefore = this.calculateStd(validValues, meanBefore);

    // ---- 步骤 F: 执行插补算法 ----
    const imputedValues = this.imputeColumn(numericData, methodId, params);
    const imputedOnlyValues: number[] = [];

    // ---- 步骤 G: 记录插补详情 ----
    for (const idx of missingIndices) {
      const imputedValue = imputedValues[idx];
      if (imputedValue !== null) {
        imputedOnlyValues.push(imputedValue);
        details.push({
          resultId,
          columnName,
          rowIndex: idx,
          originalValue: null,
          imputedValue,
          confidence: this.calculateConfidence(methodId, imputedValue, validValues),
          imputationMethod: methodId,
        });
        // 更新原数据
        data[idx][columnName] = imputedValue;
      }
    }

    // ---- 步骤 H: 计算插补后统计 ----
    const allValuesAfter = [...validValues, ...imputedOnlyValues];
    const meanAfter = this.calculateMean(allValuesAfter);
    const stdAfter = this.calculateStd(allValuesAfter, meanAfter);

    // ---- 步骤 I: 记录列统计 ----
    columnStats.push({
      resultId,
      columnName,
      missingCount: missingIndices.length,
      imputedCount: imputedOnlyValues.length,
      imputationRate: imputedOnlyValues.length / missingIndices.length,
      meanBefore,
      meanAfter,
      stdBefore,
      stdAfter,
      minImputed: imputedOnlyValues.length > 0 ? Math.min(...imputedOnlyValues) : undefined,
      maxImputed: imputedOnlyValues.length > 0 ? Math.max(...imputedOnlyValues) : undefined,
    });
  }

  onProgress(100, '插补完成');
  return { imputedData: data, details, columnStats };
}
```

#### 3.3.2 单列插补函数 `imputeColumn()`

根据方法 ID 选择对应的插补算法：

```typescript
// 第 369-403 行
private imputeColumn(
  data: (number | null)[],
  methodId: ImputationMethodId,
  params: Record<string, any>
): (number | null)[] {
  const result = [...data];
  const validValues = data.filter(v => v !== null) as number[];

  if (validValues.length === 0) {
    return result; // 无法插补
  }

  switch (methodId) {
    case 'MEAN':
      return this.imputeMean(result, validValues);
    case 'MEDIAN':
      return this.imputeMedian(result, validValues);
    case 'MODE':
      return this.imputeMode(result, validValues);
    case 'FORWARD_FILL':
      return this.imputeForwardFill(result);
    case 'BACKWARD_FILL':
      return this.imputeBackwardFill(result);
    case 'LINEAR':
      return this.imputeLinear(result);
    case 'SPLINE':
      return this.imputeSpline(result, params.degree || 3);
    case 'POLYNOMIAL':
      return this.imputePolynomial(result, params.degree || 2);
    default:
      // 需要 Python 的方法，使用线性插值作为后备
      console.warn(`方法 ${methodId} 需要Python支持，使用线性插值作为后备`);
      return this.imputeLinear(result);
  }
}
```

#### 3.3.3 均值插补实现示例

```typescript
// 第 410-413 行
private imputeMean(data: (number | null)[], validValues: number[]): (number | null)[] {
  const mean = this.calculateMean(validValues);
  return data.map(v => v === null ? mean : v);
}

// 均值计算
private calculateMean(values: number[]): number {
  if (values.length === 0) return 0;
  return values.reduce((a, b) => a + b, 0) / values.length;
}
```

```typescript

insertDetails(details: Array<{

  timePoint?: string;

  // 使用事务批量插入，提高性能
  const insertMany = this.db.transaction((items: typeof details) => {
    for (const item of items) {
      insertStmt.run(
        item.resultId,
        item.columnName,
        item.rowIndex,
        item.timePoint || null,
        item.originalValue ?? null,
        item.imputedValue,
        item.confidence || null,
        item.imputationMethod || null,
        item.neighborValues ? JSON.stringify(item.neighborValues) : null
      );
    }
  });

  insertMany(details);
}
```

## 7. 错误处理

### 7.1 常见错误及处理

| 错误 | 发生位置 | 处理方式 |
|------|----------|----------|
| `版本不存在: ${versionId}` | Service 阶段 1 | 记录状态为 FAILED，返回错误 |
| `数据为空` | Service 阶段 2 | 记录状态为 FAILED，返回错误 |
| `无效的列: ${columns}` | Service 阶段 6 | 记录状态为 FAILED，返回错误 |
| SQL 参数错误 | Repository | 抛出异常，被 Service 捕获 |
| 文件不存在 | readDataFile | 抛出异常，被 Service 捕获 |

### 7.2 错误恢复机制

```typescript
// Service 中的 try-catch-finally
try {
  // 执行逻辑...
} catch (error: any) {
  // 更新状态为失败
  this.repository.updateResultStatus(resultId, 'FAILED', error.message);
  return {
    success: false,
    resultId,
    error: error.message,
  };
} finally {
  // 无论成功失败，都清理进度回调
  this.progressCallbacks.delete(resultId);
}
```
